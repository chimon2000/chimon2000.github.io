Don't Go Hybrid, Go Cross-Platform

  • Posted on: 2015-10-01
  • By: Ryan Edge

Adaptive UI

How surprised would you be if your favorite app wasn’t native? Better yet, how surprised would you be if your favorite app was the exact same code, running on multiple platforms with just a few minor differences?

This capability captures the essence of Adaptive UI: As the orientation and size of your device change, so do its layout and elements to accommodate the needs of the user in that context. As you might surmise, the cost of implementing this type of design natively is high, and that’s why hybrid apps have such an appeal.

What is a hybrid app exactly?

A hybrid app is basically an app that uses HTML, CSS, and JavaScript while running natively on a mobile platform. This can be done by either using the internal browser (WebView) of the mobile operating system, or by compiling the code into native code for each platform. The benefits of such apps are obvious: one code base for multiple platforms, reduced development time & cost, and easy design implementation across devices. These benifits come at a dwindling cost that arguably now only amounts to performance issues (mostly where games are concerned) and a loss of native look and feel (which contradicts the Adaptive UI principles that every major mobile OS have adopted).

Hybrid apps stop short in the mobile/web sphere, and because of that have given an old term new life: cross-platform apps. While hybrid apps do lip service to the “one code base - multiple platforms” mantra, cross-platform apps make it their mission by using multiple technologies to target every operating system on every type of device. They rely on some of the same technologies as hybrid apps and then some.

Here’s a list of some of the best apps that share a universal code base, but run on multiple platforms

Build Once, Run Everywhere

These apps are the top of the cross-platform food chain, in that they target everyone everywhere. This is a short list with some heavy hitters. Apps in this category target the web, the desktop, and mobile. These heavy hitters adhere heavily to Adaptive UI principles across devices.

The List

  • Slack - Only everyone’s favorite team chat app. (PC, Mac, Linux, iOS, Android, Windows Phone, and Web)
  • Netflix - No explanation needed (PC, Mac, iOS, Android, Windows Phone, Web)
  • Wunderlist - One of the top to-do list apps since its inception. (PC, Mac, iOS, Android, Windows Phone, and Web)
  • Evernote - Pretty much where note taking starts and ends (PC, Mac, iOS, Android, Windows Phone, Web)
  • Telegram - An open source secure messaging platform. (PC, Mac, Linux, iOS, Android, Windows Phone, and Web)
  • Typetalk - Never heard of them, but a Slack - like messaging app (PC, Mac, iOS, Android)

Nailed it

These guys are less cross-platform and more hybrid. Some target desktop but for the most part are a web & mobile show. They also have little to no drop off in functionality and very little variance in experiences. Chances are if you haven’t used some of them, you’ve heard of most of them

The List

  • Twitter - You’ve definitely used it (Web, iOS, Android)
  • Instagram - You’ve definitely used it (Web, iOS, Android)
  • Linkedin - You’ve definitely used it (Web, iOS, Android)
  • Uber - You’ve definitely heard of it (Web, iOS, Android)
  • Gmail - One of the best email services to date (Web, iOS)
  • Feedly - An extremely popular app for organizing feeds (Web, iOS, Android)
  • Basecamp - One of the most popular project management tools (Web, iOS, Android)
  • Untappd - Drink responsibly (Web, iOS, Android, WP)
  • Vectr - A free, minimalistic design app (Web, Mac, PC)

The Other Guys

There’s an extensive list here, so I’ll only focus on the ones that I think have some serious value. Apps on this list target 1 platform, but still do it using cross-platform technologies. The fact that some of these guys are backed by heavy hitters should tell you everything you need to know about the future of cross-platform. I’ll keep this one short.

The List

  • Atom - Everyone’s favorite new text editor. (Mac, PC, Linux)
  • Visual Studio Code - An Atom clone for web dev (Mac, PC, Linux)
  • Pixate - An awesome free app for design and prototyping. (Mac, PC)
  • Brackets - Yet another Atom clone for web dev (Mac, PC, Linux)

Closing

From leaders to underdogs, it is evident that there are many software companies that have recognized the benefits of cross-platform application development and dedicated themselves to it as their premiere platform solution. I invite you to explore many of the cross-platform technologies that enable you to do the same!
